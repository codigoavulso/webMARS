v__WEBMARS_APP_VERSION__2026-03-19

Backend Cloud e Autenticação

  • Adicionada seleção configurável do backend cloud, com encaminhamento automático para backend local em desenvolvimento e para backmars.nfiles.top/api em produção quando não existe servidor custom definido.
  • Integrados na UI os fluxos de login, registo e criação de utilizador cloud, com suporte backend pronto para produção e feedback operacional mais claro.
  • Separada a autenticação da sincronização de projetos, para que login/registo terminem de imediato e a sync continue em background.
  • Melhorados os diálogos de login/registo com estado dentro da própria janela, tempo decorrido, cancelamento mais seguro e mensagens explícitas de atividade cloud em Mars Messages.

Melhorias no Fluxo Project e Editor

  • O fluxo principal de autoria foi consolidado em torno de Project, Editor e Execute, mantendo a disponibilidade das ações C0/Assembly conforme o tipo do ficheiro ativo.
  • A árvore de projeto ganhou defaults de expandir/recolher mais úteis, regras de abertura em read-only para libs/ e ficheiros de outros projetos, seleção múltipla, metadados/estado e indicadores de sincronização.
  • As tabs do editor passaram a restaurar corretamente sessões read-only e incluem menu de contexto com Close, Save e Save and Close.
  • As operações de gravação foram simplificadas para Save Active File e Save Project, com persistência no browser e sincronização cloud quando aplicável.

Refino de Armazenamento, Mini-C e UI

  • A quota de armazenamento do utilizador no browser foi aumentada para 1 MB, movida para configuração central e passou a ser aplicada de forma consistente.
  • O fluxo de output do Mini-C foi refinado com opção de suprimir a janela de output, abertura direta do ASM gerado e fecho automático após abrir ASM.
  • Settings cloud/servidor, traduções e comportamento dos diálogos foram alargados para suportar o novo modelo de deployment e o fluxo backend de produção.

  • O runtime ficou mais resiliente a exceções de render da UI que podiam deixar os botões desativados.
  • Foi normalizada a emissão de linhas em Mars Messages e Run I/O, reduzindo quebras em branco excessivas.

Últimos Ajustes de Paridade MARS MIPS 4.5

  • O autoload de sessão passou a reabrir código em modo Edit e a exigir Assemble explícito, evitando restauros de estado assembled obsoletos.
  • Menus e diálogos receberam ajustes finais de consistência com os padrões de interação do MARS Java.
