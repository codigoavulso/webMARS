webMARS 0.3

webMARS é uma porta JavaScript do simulador original MARS MIPS 4.5, adaptado para ser executado nativamente no navegador.

Créditos

  • Versão original em Java: Pete Sanderson, Kenneth Vollmar e colaboradores (2003-2013).
  • Porta JavaScript: Nelson Ferreira (2020-2026).

Privacidade & Licença

O webMARS é gratuito e de código aberto, sem publicidade, sem cookies e sem coleta de dados do usuário.

Distribuído sob a licença do MIT.

Status

Trata-se de um projeto em curso. Alguns comportamentos podem ser diferentes do Java MARS de desktop em casos de borda.
