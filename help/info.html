webMARS 0.3

webMARS is a JavaScript port of the original MARS MIPS 4.5 simulator, adapted to run natively in the browser.

Credits

  • Original Java version: Pete Sanderson, Kenneth Vollmar, and contributors (2003-2013).
  • JavaScript port: Nelson Ferreira (2020-2026).

Privacy & License

webMARS is free and open-source, with no advertising, no cookies, and no user data collection.

Distributed under the MIT license.

Status

This is an ongoing project. Some behaviors may differ from desktop Java MARS in edge cases.
